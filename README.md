## Скачать можно на Github

### Это учебный проект создан для сдачи домашки на курсе SkyPro

#### 1.Проект разделен на разные модули, в каждом модуле написан отдельный функционал.
#### 2. Все модули находяться в папке SRC  
#### 3. В папке Tests модули для тестировки
#### 4. base_product.py родительский модуль для класса product
#### 5. классы smartphone и lawn_grass(Трава газонная) наследники класса product
#### 6. Класс mixins печатать в консоль информацию о том, от какого класса и с какими параметрами был создан объект
